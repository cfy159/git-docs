# 1.git配置

```shell
git config --global user.name "用户名"
git config --global user.email "邮箱地址"


# 查看配置信息
git config --global --list
```

# 2.初始化git仓库

```shell
git init [仓库名]
```

# 3.配置本地SSH key

```shell
ssh-keygen -t ed25519 [-C "邮箱地址"]


cat ~/.ssh/id_ed25519.pub
```

配置ssh密钥

# 4.建立远程连接

```shell
# 绑定远程仓库，别名设为 github_ros
git remote add origin git@github.com:xxx
# 重命名分支为main
git branch -m master main
# 建立本地分支和远程分支
git branch --set-upstream-to=origin/main main


# 拉取/推送
git pull
git push
```



## 常见问题

拉取时出现无效引用：fatal: 无效引用：origin/main

```shell
# 获取远程分支信息，不会改变本地数据
git fetch origin

#查看分支信息
git branch -r
```

更新指定文件

```shell
git checkout origin/main -- <文件名>
```
